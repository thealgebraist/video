# vidlib

A unified Python library for all reusable code in this project.

## Structure
- `assemble.py`: Trailer assembly functions
- `assets.py`: Asset generation functions
- `export.py`: Export and conversion utilities
- `utils.py`: Utility classes and functions
- `torchvideo.py`: Video reference utilities
- `remote.py`: Remote text encoder utilities
- `test.py`: Test utilities
- `verify.py`: Verification utilities

## Usage
```python
from vidlib import *
```
